# Getting started 🚀

## Tags Supported

Button Group

```
#bs:buttonGroup(isVertical, classExtras?, Aria?) { }
```

Button Toolbar

```
#bs:buttonToolbar(classExtras?, Aria?) { }
```


## Example of Leaf

```
#bs:buttonGroup(false, "btn-group-sm") {
#bs:button() { First Option }
#bs:button("danger") { Second Option}
#bs:button() { Third Option}
}

```

```
#bs:buttonToolbar() {
#bs:button() { First Option }
#bs:button("danger") { Second Option}
#bs:button() { Third Option}
}

```
